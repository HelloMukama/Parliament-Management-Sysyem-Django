Problems navigating around the system?
Read through some of the hints below to guide you on where you could be facing issues

Dashboard Description
The Dashboard is a collection of graphs and charts representing data from all of your submissions. These charts each have an option of saving or printing a copy in that you could attach this copy to say a report you're drafting. Easy right! Another thing is that each of these charts updates with fresh data at page reload with all of your recently entered data.

These charts are clealy annotated showing you exactly what you need to know at first glance! Figures of say the region or category with the most or least number of submissions.

Submissions: ALL, UNRESOLVED & RESOLVED
These pages each contain a tabled list of your submissions in a very easy to understand manner. Column 1 shows Submission IDs with background colors of red, yellow and green. Green is for solved Yellow is for active / For the submissions that are actively being worked upon Red is for pending: This means that you either just submitted this or did the submission some time back but it's yet to be brought to the floor of the parliament.

All Submissions page: This shows all the submissions you've made via the system as depicted by red, yellow and green colors

Unresolved Submissions page: This shows all the unresolved submissions.

Resolved Submissions page: This shows all the resolved submissions. You can sort, search and perform amazing actions on the tables holding these submissions

Add new
This page is where you can make a new submission. After making a new submission, you'll be redirected to the all submissions table in that you can view that submission you just made.

Viewing a submission
When you click on any submission inside of the tables, you'll be taken to the details of that particular submission. Here, you'll be able to edit, leave a comment and edit your comments.

NB: It's only the Speaker that will be able to view these as the submissions you make are really to the speaker's office.
What does this mean? This then means that it's the speaker that can see your submission, and also leave a comment. When the speaker adds a comment, you'll get an email notification of this comment and from here, you'll be able to log onto your account and react to this comment.
Feel free to checkout more that can be done with the comments. Since forexample, in case a comment is inappropriate, you can report it to the administrators

The Statistics page
This page shows tables and charts.

TablesFor the tables, these show totals from different arrangements. Each thing is a clear as can be and therefore, it saves your time when you're viewing these tables.

Charts These charts' description is the same as that offered for the dashboard charts.

Accounts and Profiles

My profile
You can view your profile details and also edit these details.

Settings
Settings is a page where you can perform system changes like changing of the password of your sysem.
